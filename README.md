# Overview

[OpenSCAD](https://openscad.org/) model of the clavichord depicted in the 1479 intarsia from the
Studiolo of Ducal Palace at Urbino. Based on dimensions from [The Urbino Clavichord
Revisited](https://harpsichords.weebly.com/uploads/2/5/0/1/25019733/verbeek_magnano_sep_2011.pdf) by
[Pierre Verbeek](https://harpsichords.weebly.com/).
